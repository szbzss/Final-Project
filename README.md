# Final-Project
Hoverboard Robotic Furniture: A-Guided Adaptive DWA Navigation
This repository contains the source code, simulation environments, and control logic for the "Hoverboard Robotic Furniture" project. The system integrates an ESP32-driven 12-TFmini ToF sensor array with a hybrid A*-Guided Adaptive Dynamic Window Approach (A-DWA) for autonomous navigation in complex indoor environments.

🚀 Features
Adaptive DWA: Dynamic weight reallocation to escape local minima (U-shaped obstacles).
Hardware-Timed Perception: 100Hz synchronized polling for a 12-sensor LiDAR array via ESP32.
ROS 2 Integration: Full support for Gmapping SLAM and navigation stacks.
Simulation-Ready: Pre-configured Webots/Gazebo environments for testing.
